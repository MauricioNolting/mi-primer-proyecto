    • JAVASCRIPT

    JavaScript es un lenguaje de programación ampliamente utilizado en el desarrollo web para crear interactividad y dinamismo en las páginas. Es interpretado por los navegadores y permite a los desarrolladores manipular el contenido de una página, responder a eventos del usuario y comunicarse con servidores. Una de sus características distintivas es su capacidad para modificar el contenido HTML y el estilo CSS en tiempo real.

    Función

    Se utiliza para crear efectos visuales, validar formularios, realizar animaciones, gestionar peticiones asíncronas y desarrollar aplicaciones web complejas.

    CSS

    CSS es un lenguaje utilizado para controlar la presentación y el diseño de las páginas web. Permite definir cómo se mostrará el contenido HTML, estableciendo propiedades como colores, fuentes, márgenes y tamaños. A diferencia de HTML, que se enfoca en la estructura y el contenido, CSS se centra en la apariencia visual. Una característica clave de CSS es su capacidad para aplicar reglas de estilo de manera jerárquica.

    Función

    Permite la creación de diseños consistentes y flexibles en todo el sitio web. Además, el concepto de "cascada" se refiere a cómo las reglas se aplican en función de su especificidad y posición en el documento.

    HTML

    HTML es el lenguaje de marcado utilizado para crear la estructura y el contenido de las páginas web. Aunque HTML originalmente estaba diseñado para definir la estructura del contenido, con las versiones más recientes se han introducido atributos y elementos que permiten una mayor semántica y funcionalidad, como la reproducción de video y audio, la inclusión de gráficos vectoriales y la implementación de formularios más avanzados.

    Función

    Define la semántica de los elementos presentes en una página, como encabezados, párrafos, imágenes, enlaces y formularios. Utiliza una estructura de etiquetas anidadas para organizar y describir la información.
